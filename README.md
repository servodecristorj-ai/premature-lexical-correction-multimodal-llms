# premature-lexical-correction-multimodal-llms
Case study on premature lexical correction in multimodal LLMs. Documents failure in Claude, Meta AI, Gemini, ChatGPT auto-correcting 'subscreveu'→'subiu' before image context. Proposes: uncertainty-aware correction, deferred judgment, lexical calibration. Includes tests, PDF evidence, technical report. arXiv cs.CL 2026.
