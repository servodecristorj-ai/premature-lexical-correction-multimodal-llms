# Premature Lexical Correction in Multimodal LLMs via Deferred Context Grounding

A systemic evaluation of how autoregressive decoding priors cause greedy over-correction on rare but grammatically valid inputs before cross-modal processing occurs.

---

## 📌 Abstract
This repository documents a systemic failure mode in state-of-the-art multimodal Large Language Models (LLMs): **premature lexical correction**. When presented with low-probability but grammatically valid textual sequences, models apply aggressive auto-correction based entirely on text-only priors before performing cross-modal grounding with images submitted in subsequent conversational turns. 

We demonstrate this behavior through a controlled case study using the Portuguese construction *"O gato subscreveu na telha"* (The cat signed/subscribed on the tile).

## 🔬 Methodology & Core Experiment
The experiment tests model resilience against deceptive linguistic clichés. While the sequence *"O gato subiu no telhado"* (The cat climbed on the roof) is massivamente present in training sets, we replace the verb with a rare, highly formal, yet fully valid term:

1.  **Turn 1 (Text-only):** *"O gato subscreveu na telha"*
2.  **Turn 2 (Visual Token):** An image depicting the cartoon character *Felix the Cat* physically writing and signing a letter on top of a tiled roof.

### Tested Systems (July 2026)
*   **Claude 3.5 Sonnet** (Anthropic) - *Main Case Study*
*   **ChatGPT-4o** (OpenAI)
*   **Gemini** (Google)
*   **Meta AI** (Meta)

All models exhibited 100% reproducibility of the failure mode during Turn 1, forcefully claiming the phrase was semantically invalid or a typo until the visual grounding anchor was introduced in Turn 2.

## 🛠️ Proposed Mitigations
As detailed in the technical report, three architectural updates are proposed:
1.  **Uncertainty-Aware Correction:** Consultative hypothesis triggers instead of silent text corruption.
2.  **Deferred Judgment:** Postponing lexical normalization until the multimodal input graph is fully loaded.
3.  **Lexical Calibration:** Reducing probability penalties for valid dictionary terms within complex syntax.
