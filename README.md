Premature Lexical Correction in Multimodal LLMs via Deferred Context Grounding
Status: Work in progress | Paper: PDF in preparation - arXiv submission planned for Jul 2026
Abstract
This repository documents a systemic failure in multimodal LLMs: automatic lexical correction applied to text before processing attached images, causing false positives on rare but semantically valid terms.
We present a case study reproduced across 4 production LLMs and propose three mitigation strategies.
Experimental Evidence
We tested the case on 4 major multimodal LLMs. All exhibited premature correction.

Proposed Mitigations
Uncertainty-Aware Correction: Query user when confidence in "typo" hypothesis is low.
Deferred Judgment: Postpone lexical correction until multimodal context is complete.
Lexical Calibration: Reduce penalty for rare but grammatically valid terms.
